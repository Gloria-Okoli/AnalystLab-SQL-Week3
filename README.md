# AnalystLab Africa — Week 3: SQL & Data Querying

## 📌 Description
SQL analysis tasks completed during Week 3 of the AnalystLab Africa 
Data Analytics Internship (Batch B, June–August 2026). 
This week focused on building SQL querying skills using two real-world datasets.

## 🛠️ Tools & Technologies
- Microsoft SQL Server Express
- SQL Server Management Studio (SSMS)
- Azure Data Studio

## 📂 Datasets
| Dataset | Source | Description |
|---------|--------|-------------|
| Chinook Database | GitHub (lerocha/chinook-database) | Music store data with 11 tables |
| Sales Dataset | Kaggle (kyanyoga/sample-sales-data) | Product sales data across countries |

## 📊 SQL Concepts Covered
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- Aggregate Functions (SUM, AVG, COUNT)
- INNER JOIN, Multiple JOINs
- Subqueries
- Window Functions (RANK, OVER, PARTITION BY)
